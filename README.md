# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/6d258b99-4e63-4aa9-a6b7-0676cd3a8a44

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

# 🧧 随礼账本系统 (Gift Giving System)

一个基于 React + Express + Supabase 开发的个人随礼管理系统。

### 🚀 快速启动
1. 克隆项目：`git clone https://github.com/DavidH0811/gift_giving_system.git`
2. 安装依赖：`npm install`
3. 配置环境：将 `.env.example` 重命名为 `.env` 并填入你的 Supabase Key。
4. 运行项目：`npm run dev`
